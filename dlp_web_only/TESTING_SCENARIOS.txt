CYBER SENTINEL - TESTING SCENARIOS
===================================

These scenarios will help you thoroughly test and demonstrate all features.

SCENARIO 1: Initial Setup & Configuration
------------------------------------------
Goal: Configure the system for first use

Steps:
1. Start server: python dlp_web.py
2. Open browser: http://127.0.0.1:5000
3. Enter API key when prompted (from dlp_api_key.txt)
4. Go to Configuration page
5. Verify monitored path: ./monitored_data
6. Add authorized destination: internal_server.com
7. Save configuration
8. Verify: Toast notification "Configuration saved"

Expected Result:
✓ System configured successfully
✓ Monitored path set correctly
✓ Ready to scan files
